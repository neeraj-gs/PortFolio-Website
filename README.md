# Neeraj GS - Portfolio

A modern, responsive portfolio website showcasing my work as a Full Stack Developer & AI Solutions Engineer.

**[View Live Site](https://neeraj-gs-portfolio.vercel.app)**

## About

This portfolio highlights my professional experience, technical skills, and featured projects. It includes sections for work experience, skills, project showcases with video demos, and a contact form.

## Features

- Responsive design optimized for all devices
- Dark/Light theme toggle
- Interactive project cards with embedded video demos
- Smooth scroll animations using ScrollReveal
- Contact form powered by EmailJS
- Clean, modern UI with custom CSS

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (ES6+)
- [RemixIcons](https://remixicon.com/)
- [ScrollReveal](https://scrollrevealjs.org/)
- [EmailJS](https://www.emailjs.com/)

## Sections

- **Home** - Introduction and quick links
- **Experience** - Professional work history at Penomo
- **About** - Background and specializations
- **Projects** - Featured work with live demos
- **Skills** - Technical expertise across Frontend, Backend, Blockchain, Databases, DevOps & AI
- **Contact** - Get in touch form and social links
